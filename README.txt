Markov Decision Processes

Instructions
=============

- Download/Clone code repository from: https://github.com/habbes/ml-mdp-rl-experiments-omscs

- Install dependencies:

* Python 3
* Anaconda
* Jupyter Notebook
* Matplotlib
* Seaborn
* OpenAI Gym
* pymdptoolbox : https://pymdptoolbox.readthedocs.io/en/latest/

- Run the code

Run the `mdp.ipynb` in a Jupyter notebook. The main code is there.
The remaining files are helper libraries and classes and
implementations of some algorithms and MDPs
